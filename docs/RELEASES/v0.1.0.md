# v0.1.0 - Codex 日报公开 Alpha

`v0.1.0` 验证一件事：能否从本地 Codex 会话稳定生成有价值、可解释的每日“数据、成果、层级与建议”。

## 包含

- macOS Apple Silicon / Intel 单二进制 `aci`。
- 只读流式扫描 Codex JSONL，固定本地 19:00-19:00 日报窗口。
- 会话、消息、工具、Skill、Subagent、活跃时间和 Token 指标。
- 工作区间、任务、成果、验证状态、五维证据评分与当日 L1-L4。
- 经授权的 `codex exec --ephemeral` 语义分析；失败时自动降级。
- 无 JavaScript、无远程资源的自包含 HTML 与本地历史索引。
- 19:00 LaunchAgent、最近 7 日补偿、幂等 revision 和 manifest-owned 清理。

## 不包含

Claude Code、OpenCode、WorkBuddy、Qoder、周/月报、常驻 daemon、localhost API、React Dashboard、SQLite、人工纠错、`.app`、自动升级、Apple Developer ID 签名或公证。

## 安装与校验

```sh
curl -fsSL https://github.com/PANGKAIFENG/ai-collaboration-insights/releases/download/v0.1.0/install.sh | sh
aci doctor
aci report --no-ai --open
```

安装器按架构下载二进制，并在替换前校验 Release 中的 `checksums.txt`。GitHub Release 同时提供 build provenance attestation。

## 隐私边界

所有源日志保持只读。项目不上传会话、代码或报告，不保存 API Key，也不包含 telemetry。AI 分析只在明确授权后调用本机 Codex 登录态，并发送经过脱敏和大小限制的最小分析包。

## 已知限制

- 当前仅验证受支持的 macOS 环境。
- 二进制未签名、未公证，首次运行可能触发 macOS 安全确认。
- 任务名称、成果和评分是证据驱动的推断；证据不足会明确显示不可用或较低置信度。
- 静态日报不支持交互式纠正和跨日趋势。

完整验证证据见 `docs/TESTS/v0.1.0-verification.md`。
